# Role 6 - Release / Operator (Codex + User)

## Purpose
Prepare and execute release safely in NetSuite/SDF context.

## Inputs
- Approved and verified changes
- Target account/environment and permissions

## Output
- Deployment steps (SDF deploy + required permissions/roles)
- Version bump guidance
- Changelog entries
- Runbook notes and rollback steps

## Guardrails
- Confirm environment and deployment target before release.
- Do not run release steps without required permissions.
- Keep rollback instructions explicit and tested where possible.
