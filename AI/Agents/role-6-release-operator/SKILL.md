# Role 6 - Release / Operator

## Purpose
Prepare and execute release safely.

## Required Output
- SDF deployment steps and required permissions/roles
- Version bump guidance
- Changelog entries
- Runbook and rollback steps

## Guardrails
- Confirm target account/environment before release.
- Do not run release actions without required permissions.
- Keep rollback instructions explicit.
